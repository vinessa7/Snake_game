# Snake Game

A simple Python `turtle`-based Snake game.

## Repository

Clone the repository using the Git URL:

    git clone git@github.com:vinessa7/Snake_game.git

## Requirements

- Python 3.8+ (Python 3.10 or later recommended)
- Standard library `turtle` module (included with CPython)
- No third-party packages required

## Files

- `snake_game.py` - Main game script
- `apple.gif` - Food sprite used by the game
- `background.gif` - Background image used by the game

## Setup

1. Make sure Python 3 is installed:

   python3 --version

2. Change into the project directory:

   cd Snake_game

3. Run the game:

   python3 snake_game.py

## How to Play

- Use the arrow keys or `W`, `A`, `S`, `D` to move the snake
- Eat the apple to grow longer and increase your score
- Avoid colliding with the border or the snake's body

## Notes

- The game uses the `turtle` graphics library and will open a new window when launched.
- If the window does not appear, ensure you run the script in a desktop environment with GUI support.
